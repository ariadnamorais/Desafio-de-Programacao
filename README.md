### Como utilizar

👉 Como utilizar o Login

1º Ao executar o sistema, insira o Usuário admin e a Senha 1234

2º Clique em entrar.


<img src="https://media.giphy.com/media/dAq7mpXtOczGCBAXlh/giphy.gif" width="700" height="300" />

👉 Como utilizar a tela de Usuários

1º Após logar e estar no menu principal, selecione o botão Usuários;

2º Nesta tela, você pode informar os dados do novo usuário e clicar em salvar;

3º Você pode excluir usuário, para isso, clique sobre o usuário que deseja excluir e clique em excluir;

4º Você pode editar as informações do usuário, para isso, clique sobre o usuário que deseja editar, clique em alterar, mude as informações necessárias e clique em salvar;

5º O botão de limpar, serve para limpar os campos de nome, login e senha;

6º O botão voltar, serve para voltar para a página principal.

<img src="https://media.giphy.com/media/IeXg6rTXMvBy4RJAQh/giphy.gif" width="700" height="300" />

👉 Como utilizar a tela de Sobre

1º Para visualizar o sobre, no menu principal, clique em sobre;

2º O sobre irá mostrar um pouco sobre o processo de desenvolvimento;

3º O botão voltar serve para voltar para o menu principal.

<img src="https://media.giphy.com/media/JD5jvnnsvQRJ3ewlG1/giphy.gif" width="700" height="300" />

👉 Como utilizar a tela de Anúncios

1º Para cadastrar um novo anúncio, insira os dados: Nome do anúncio, Cliente, Investimento por dia e datas iniciais e finais (Atenção as datas! É possível inserir data utilizando apenas ponto. Ex.: 07.04.2021 ou 07.04)

2º Logo após informar os dados, clique em salvar. As informações serão preenchidas na tabela.

3º Para calcular as máximas (visualizações, cliques e compartilhamentos), clique sobre o anúncio que deseja calcular e clique em Calcular Máximas. O resultado aparecerá na tabela. (Obs.: Ao lado do botão calcular máximas, existe um botão de interrogação para casos de dúvidas).

4º O botão limpar serve para limpar os campos digitáveis da tela.

5º O botão excluir serve para excluir um anúncio. Basta clicar no anúncio que deseja excluir e clicar em excluir.

6º O botão voltar serve para voltar para a tela principal.

7º O campo de valor total de investimento irá ficar atualizando conforme for incluindo ou excluindo anúncios.

8º Na tabela é possível ver os resultados do investimento como um relatório.

### Pré-requisitos 

👉Antes de começar, você vai precisar ter instalado em sua máquina as seguintes ferramentas:
[Git](https://git-scm.com), [Jdk 11](https://www.oracle.com/br/java/technologies/javase-jdk11-downloads.html).

👉Além disto é bom ter um editor para trabalhar com o código e compilar a aplicação.
[NetBeans IDE 11.1](https://netbeans.apache.org/download/nb111/nb111.html)()


### 🎲 Clonando o repositório

```bash
# Clone este repositório
$ git clone https://github.com/ariadnamorais/Desafio-de-Programacao

# Acesse a pasta do projeto no terminal/cmd
$ cd desafio-de-Programacao

```
### 🎲 Para rodar o projeto da parte 2 do desafio
- A pasta do projeto está neste repositório e na pasta Calculadora
- Acesse o NetBeans IDE 11.1
- Clique em File -> Open Project
- Selecione o projeto Calculadora (Esse projeto é a parte 2 do desafio)
- Clique em Open Project
- Clique com o botão direto do mouse sobre o projeto
- Clique em RUN (Aqui o projeto irá abrir)
- Na tela de login, utilize o usuário: admin e senha: 1234
